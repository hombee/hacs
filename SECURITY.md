# Security policy

## Supported versions

Only the latest published Hombee Air HACS release receives security fixes.

## Reporting a vulnerability

Report suspected vulnerabilities privately through GitHub security advisories on
this repository. Do not open a public issue for a security problem.

Include:

- affected version or commit
- Home Assistant version
- steps to reproduce
- expected impact
- any logs or packet captures that do not expose customer secrets

We will acknowledge valid reports as quickly as possible and coordinate a fix
through a private advisory before public disclosure.
